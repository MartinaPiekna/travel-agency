# Travel Agency

- Created during a short Hackathon as a simple prototype of a space travel agency website.
- The Travel Agency project is primarily a visual concept that allows me to exercise my creativity.

## How to execute a code.

- You can launch the preview in a browser using the VS Code IDE's LiveServer extension.

_Note_:

- This project is still ongoing due to a lack of time.
- The website is not currently connected to any server or database and has only basic front-end functionality, including the primary dummy content.
